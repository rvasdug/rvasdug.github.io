### Description
This session will cover what you need to know to get started using the Microsoft Entity Framework: what the difference is between Code First and Model First, how to build out a model using either pattern, generating an edmx file, the basics of database context, how entities are generated and how to use them and last but not least, how to write CRUD Operations to perform basic operations with data using C# and Entity SQL.
