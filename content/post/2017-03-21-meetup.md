### Description
This month we will focus on part 2 of our 3 part series on JavaScript frameworks and will get an introduction to Angular2.

In this talk, we will look at a couple of different ways to quickly get started building Angular 2 applications using Visual Studio Code. First, we will use the Angular Self-Executing Bundle to create a simple Angular 2 application using basic JavaScript functions. Then we will progressively move on to using tools like NPM, Yeoman and TypeScript to build sophisticated applications in Angular 2.

### Speaker: Sidney Andrews
[Sidney Andrews](https://x.com/@sidney_andrews) is an Azure MVP and MCT based out of Richmond, VA. He has a long history developing XAML and C# applications. Recently, Sidney has changed gears to focus on Node and Angular solutions using managed service platforms on Azure. Sidney is also the author of the Introduction to Angular 2 Development course on edX.org.
