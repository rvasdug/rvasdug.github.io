### Description
Rob Tompkins, of the Apache Commons Project, will talk about how he started doing open source Java development, and how he made it up the ranks inside of the Apache Commons Project to the point where he is now cutting releases of the ubiquitous Commons-Lang3 Java component.

### Speaker: Rob Tompkins
Rob Tompkins is a Java/DevOps developer at Capital One. He also is a committee member for the Apache Commons Project. His specialties are in CICD streamlining, automated reasoning, and edit distances.
