### Description
Are you an existing .NET developer who’s trying to keep up with the brisk pace of updates? Perhaps you are someone who’s never developed in .NET before and is curious about the framework. In this talk, we will look at the latest version of .NET and talk about new features, the roadmap, and updates to the C# programming language. We will go over some really cool stuff such as the ability to compile for places where .NET is not installed, pulling tools from NuGet like you can do with NPM, asynchronous console applications, pattern matching in C#, and even the new tuples and discard syntax.

Even if you have worked in .NET you entire career, you will be surprised how many new platform and language features can make it easier for you to develop applications every day.
